## 3.基于pybind11编译py文件为.so/.dll共享库（python集成到c++）

**1.把原来的python fastapi封装为函数py，如related_storage.py；**

**2.写一个c++.cpp来插件封装，注意：**

```
py::module algo = py::module::import("related_storage");
```

 里的import("related_storage")一定是你的py文件名！

**3.CMake 构建 DLL（CMakeLists.txt）**

如：

```
cmake_minimum_required(VERSION 3.15)

project(storage_plugin)

set(CMAKE_CXX_STANDARD 17)

find_package(pybind11 REQUIRED)

add_library(related_storage SHARED plugin.cpp)

target_link_libraries(related_storage PRIVATE pybind11::embed)
```

**4.Windows编译**

```
mkdir build
cd build
cmake ..
cmake --build . --config Release
```

（注：cmke ..可能会遇到**pybind11 没被 CMake 找到**的情况，这时候运行：

```
python -m pybind11 --cmakedir
```

写一个测试cpp，如：

```
#include <windows.h>
#include <iostream>

typedef const char* (*AlgoFunc)(const char*);

int main()
{
    HMODULE lib = LoadLibraryA("related_storage.dll");

    if (!lib)
    {
        std::cout << "DLL load failed\n";
        return -1;
    }

    AlgoFunc func = (AlgoFunc)GetProcAddress(lib, "related_storage");

    if (!func)
    {
        std::cout << "function not found\n";
        return -1;
    }

    const char* json = R"(
{
  "target_items":[
    [
      {"id":"A","qty":10}
    ]
  ],
  "history_records":[],
  "min_support":1,
  "max_recommendations":3,
  "time_decay":0
}
)";

    const char* result = func(json);

    std::cout << "Result:\n" << result << std::endl;

    return 0;
}
```
